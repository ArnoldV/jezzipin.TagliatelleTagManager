Tagliatelle Tag Manager

Automated tag generation and management for Umbraco

What's all the fuss about?

Tagliatelle is a tag property editor for Umbraco 7+.

It allows for automatic page generation based upon tags entered into a property editor making it ideal for blog tag management or list functionality.

Get stuck in

To use Tagliatelle:

  1. Create a new datatype using the Tagliatelle Tag Editor datatype

  2. Select the content node you wish to be the parent of generated tag pages.

  3. Select the document type you wish to use for your tag pages.

  4. Add your datatype to the document type of your choosing.

  5. Create a content node based upon your new document type. In the Tagliatelle property editor, add a number of tags by simply typing the tag names you wish to create.

  6. Save the content page to generate your new tags.

You can also select and assign existing tags using the Tagliatelle Tag Editor.
